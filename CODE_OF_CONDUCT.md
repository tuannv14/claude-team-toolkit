# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1 as its code of conduct.

All contributors and participants are expected to follow this standard when interacting in any project space (issues, pull requests, discussions, email, social channels referencing the project).

## Reporting

To report a concern about behavior in this project, email **96.tuan.nv@gmail.com** privately. Reports are kept confidential. You will receive an acknowledgement within 72 hours.

See [SECURITY.md](SECURITY.md) for separate guidance on reporting security vulnerabilities.
